# Bike Sales Dashboard Project

This project is an interactive Bike Sales Dashboard developed in Microsoft Excel to analyze customer purchasing behavior and identify key business insights. The project demonstrates the complete data analysis process, including data cleaning, transformation, visualization, and dashboard creation.

The dataset contains customer information such as age, gender, marital status, education, occupation, income, home ownership, number of cars owned, commute distance, region, and bike purchase status. The data was cleaned and prepared to ensure consistency and accuracy before analysis.

Using Pivot Tables and Pivot Charts, different business questions were explored to understand customer buying patterns. Interactive Slicers were added to allow users to filter the dashboard dynamically by different categories, making the analysis more user-friendly and informative.

The dashboard provides valuable insights into how demographic and financial factors influence bike purchasing decisions. It enables users to compare customer groups, identify purchasing trends, and make data-driven decisions through an easy-to-use visual interface.

* Clean and organize raw data for analysis.
* Create meaningful Pivot Tables and Charts.

This project demonstrates practical Excel skills commonly used in data analysis roles and showcases the ability to transform raw data into actionable business insights through effective reporting and visualization techniques.
